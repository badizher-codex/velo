The browser that
works for you, not against you

VELO is a privacy-first Windows browser with zero telemetry, local AI threat detection, DNS‑over‑HTTPS and fingerprint protection — all built-in, no extensions needed.

Windows 10/11 x64  ·  Requires WebView2 Runtime (included in Windows 11)

🛡️

Zero telemetry

VELO never phones home. No crash reports, no usage analytics, no update pings. Your traffic stays between you and the sites you visit.

🤖

Local AI security

Threat analysis runs on your machine with Ollama or offline heuristics. Claude API is optional. No request data leaves your device.

🔓

Free & open source

Every line of code is auditable. Licensed under AGPLv3 — you can inspect, fork and improve it freely.

What's included

Everything built-in, nothing phoning home

No extension store, no upsells. Every privacy and security feature ships in the box.

🧬

Fingerprint Protection

Injects noise into Canvas, WebGL, AudioContext and hardware APIs so trackers can't build a profile from your browser fingerprint.

📡

DNS-over-HTTPS

All DNS queries are encrypted via Quad9, Cloudflare, NextDNS or a custom endpoint. Your ISP never sees what you look up.

🚫

Tracker Blocker

250+ bundled tracker domains blocked at the request level. No list to enable — it's on by default from first launch.

📦

Identity Containers

Isolate different browsing contexts — work, personal, shopping — so sites can't cross-track you across sessions.

🔒

TLSGuard

Enforces HSTS preloads and verifies Certificate Transparency logs to catch certificate misissuance in real time.

📥

DownloadGuard

Blocks drive-by downloads, executable burst attacks and suspicious cross-origin binary transfers before they reach disk.

🍪

Cookie Wall Bypass

Automatically dismisses GDPR consent banners by injecting the correct accept/reject signals before the page renders.

📡

WebRTC Leak Prevention

Forces relay-only mode or disables WebRTC entirely so your real IP address is never exposed, even on VPN.

🗝️

Password Vault

AES-256 encrypted vault protected by a master password. Uses PBKDF2 with 310,000 iterations. The master password is never stored.

📖

Reader Mode

Strip ads, banners and clutter from any article with F9. Clean reading experience, zero trackers loaded.

☠️

Malwaredex

A collectible threat tracker with 43 threat types across 6 categories — know exactly what VELO blocked and why.

🌑

Dark theme only

VELO is dark by design. No eye-straining light mode, no theme toggles, no distractions. Just a clean, focused interface.

Under the hood

6-layer security pipeline

Every web request passes through all six layers before the page loads. Each layer can block, warn or pass through.

1
WebView2 Intercept
Captures every request via WebResourceRequested (<100ms budget)
2
DoH Engine
Resolves DNS over HTTPS via Quad9, Cloudflare or custom provider
3
Security Guards
Blocklist → TLS → Request → Container checks
4
AI Security Engine
Cache → Local rules → AI adapter (3s timeout, never blocks fast path)
5
Cookie Wall Bypass
ConsentInjector runs before DOMContentLoaded via script injection
6
UI Result
SecurityPanel appears only on WARN or BLOCK — no noise for clean pages
